## When to Use

- Reviewing pull requests for WooCommerce
- Checking code against WordPress/WooCommerce standards
- Automated code quality assessments
- Identifying common violations
- Ensuring consistent code style

## Key Features

- **Violation Detection**: Comprehensive list of critical issues to flag
- **Standard References**: Links to detailed documentation
- **Output Format**: Structured reporting for violations
- **Cross-Skill Integration**: References backend-dev, copy-guidelines, and dev-cycle skills
