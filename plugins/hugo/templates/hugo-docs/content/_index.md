# Welcome

This is a Hugo documentation site built with the Hugo Book theme.

## Features

- **Search**: Full-text search across all documentation
- **Navigation**: Nested sidebar with collapsible sections
- **Table of Contents**: Auto-generated TOC for each page
- **Git Integration**: Last modified dates from Git history
- **Markdown**: Full markdown support with code highlighting
- **Responsive**: Mobile-friendly design
