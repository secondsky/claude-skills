# Playwright Testing Skill

End-to-end testing with cross-browser support, visual regression, and powerful debugging.

## Overview

Expert knowledge for E2E testing with Playwright including browser automation, network mocking, visual testing, and trace viewer debugging.

## Key Features

- Cross-browser (Chromium, Firefox, WebKit)
- Auto-wait (no flaky tests)
- Powerful semantic selectors
- Network interception and mocking
- Visual testing (screenshots, videos)
- Trace viewer for debugging
- TypeScript-first API

## When to Use

- End-to-end testing
- Cross-browser testing
- Visual regression testing
- API testing
- Authentication flow testing
- User journey testing

## Quick Start

```bash
# Install
bun add -d @playwright/test
bunx playwright install

# Run tests
bunx playwright test

# Debug mode
bunx playwright test --debug

# UI mode (interactive)
bunx playwright test --ui
```
