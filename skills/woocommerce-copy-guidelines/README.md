# WooCommerce Copy Guidelines Skill

Production-ready skill for writing consistent UI text and copy in WooCommerce.

## Auto-Trigger Keywords

- WooCommerce UI text
- WooCommerce copy guidelines
- sentence case
- title case
- button text
- label text
- UI copy standards
- user-facing text

## When to Use

- Writing button labels
- Creating form field labels
- Adding notification messages
- Writing menu item text
- Creating headings in UI
- Documenting user-facing features

## Key Features

- **Sentence Case Rules**: Clear guidelines for capitalization
- **Exception Handling**: When to use capitals (proper nouns, acronyms, brands)
- **Element-Specific Examples**: Buttons, labels, menus, headings, messages
- **Quick Reference**: Lookup table for common patterns

## Token Efficiency

- Estimated savings: ~60% vs style guide research
- Errors prevented: Inconsistent capitalization, title case misuse
- Coverage: All user-facing text types
